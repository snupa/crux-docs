Crux comes in a simple, complete package that contains several components. You may choose to install them all or only the ones you believe you’ll need. Don’t worry, you’ll be able to add components at any time.

What is it and what it does?

Angular JavaScript watcher, concatenates the JavaScript files into a single build file, with custom include patterns and environment-specific configurations.

Angular Template watcher, caches the HTML contents of each template into a single JavaScript file, used by Angular at runtime to fetch the views.

Commonjs watcher, uses browserify & aliasify to provide commonjs require() functionality, equipped with dynamic require paths

Less watcher, used to easily compile less files to a CSS output.

Sass watcher, same as less, but for scss files.

Any kind of front-end templates (html, swig, etc), that will be cached under a window variable (ex: window['views'][filePath] = htmlContent) - easily cache client-side templates.
