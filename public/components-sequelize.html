What is it?

Wrapper over Sequelize's API, to easily define relationships between models (in a single file, with no other dependencies)

See API Documentation.

What it does?

Auto-load model definitions and their relationships

Database synchronization options (synchronize the model layer, or full database reset)

Loading of custom SQL files containing environment-specific SQL queries

Custom column types, wrapped over original MySQL types (ex: Seq.JSON will store objects stringified, and parse them on load time)

Model mocking functionality
