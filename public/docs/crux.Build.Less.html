Class: Less

crux.Build. Less

new Less(config)

This is the LESS build compiler. It will compile any .less files configured. It uses less@2.2 npm package to do so.

Parameters:
Name Type Description
config Object

the default process configuration object

Properties
Name Type Argument Default Description
extension String <optional>
 .less

file extension of our less files.
path String <optional>
 front/less

file path used for less sources, relative to the project's root folder
entry String <optional>
 style.less

the main less file to be compiled
output String <optional>
 public/css/style.css

the output file used to compile the less entry file
compressed Boolean <optional>
 false

should the generated CSS be compressed
files Array <optional>
 []

allows the compilation of additional .less files. These are in format of { "fileName" : "outputFile"}. Note: fileName is relative to the [path] config and outputFile is relative to the [output] config,
includes Array <optional>

additional file paths to be included while compiling the less sources
options Object <optional>

additional less options to be passed on compile time

<static> buildFiles(done)

Performs the less build process over all the configured additional files.
Note as most projects tend to have a single less file, this is called when the configuration object has the files option set. This will loop over each file object and compile it.
