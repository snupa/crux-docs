crux.Build. Interface

new Interface()

This is the interface every build process must extend. The crux build component will not run with processes that do not extend this.
Although it provides common functionality, its methods may be overridden by any custom components.
Note for a better visual representation of the API doc, we've excluded the @interface mark, but it is the Build Process Interface
