Meet Crux

The ultimate development framework

We’ve all gone through the daunting task of setting up environments and components when we get started. Development comes somewhat naturally; toying with settings just as we want to work does not. So we’ve set out to create the ultimate framework to make developers’ life so much easier.

And thus Crux was born: a modular, simple and intuitive suite that will allow you to get coding in no time. And yes, it is Open Source
